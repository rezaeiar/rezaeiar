A scalable asset management platform designed around large media libraries and modern frontend architecture.

**Stack**

`React` · `TypeScript` · `TanStack Query` · `Tailwind CSS`

**Engineering Challenge**

Building reusable upload flows, scalable state management, optimistic UI updates, and responsive asset rendering.

**Why it matters**

A visual editor for designing business workflows through composable nodes and interactive connections.

**Stack**
